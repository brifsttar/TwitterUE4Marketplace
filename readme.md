# Fab for Unreal Discord Bot

Bot (actually, script + webhook) that posts a Discord message for each new package added to the [Fab (Unreal Channel)](https://www.fab.com/channels/unreal-engine).

The bot runs on the [Fab for Unreal](https://discord.com/invite/wsPDQpxwme) Discord server.

## Features

* Posts for each new Unreal Engine product released on Fab
* Blacklist to reduce noise (e.g., AI, music)
* Dedicated channel for new Free product releases
  * Also supports *Free for two weeks* with a custom notification

## Legacy

This repo used to be a Twitter bot that fed @UnrealMarket on Twitter.
